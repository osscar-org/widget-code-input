
widget_code_input
=====================================

Version: |release|

A Jupyter widget to allow input of a python function, with syntax highlighting.


Quickstart
----------

To get started with widget_code_input, install with pip::

    pip install widget_code_input

or with conda::

    conda install widget_code_input
